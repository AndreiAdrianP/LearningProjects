# LearningProjects

##  Repository Overview

This repository documents the hands-on implementation and simulation of several fundamental **Creational Design Patterns** using modern C++ (C++17/20). The primary goal of this exercise was to gain deep, practical expertise in pattern implementation, focusing on **deterministic behavior** and constraints typical of **embedded systems** (e.g., MISRA C++ guidelines).

All projects are structured with a clean separation between libraries and client applications, managed via **CMake**.
